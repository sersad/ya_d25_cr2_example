## 📋 Формулировка 4 задачи
База данных автомобильного каталога имеет следующую структуру:

**Таблица Objects (Автомобили)**  
`id, object, type_id, measure, config_id`

**Таблица Types (Классы автомобилей)**  
`id, type`

**Таблица Config (Комплектации)**  
`id, configuration`

*Поле `measure` в данном контексте обозначает мощность двигателя (в л.с.).*

**Ввод:** имя файла базы данных, класс автомобиля, название комплектации.  
**Вывод:** названия автомобилей введённого класса с указанной комплектацией без повторений, по одному в строке, отсортированные по убыванию мощности (`measure`), затем по алфавиту.

---

### 🗃️ SQL для создания и наполнения базы
**Используйте vehicles.db из репозитория**

Или скопируйте этот код и выполните в любом SQLite-клиенте (или через `sqlite3` в терминале), чтобы получить файл `vehicles.db`.


```sql
CREATE TABLE Types (
    id   INT PRIMARY KEY,
    type VARCHAR(50)
);

CREATE TABLE Config (
    id            INT PRIMARY KEY,
    configuration VARCHAR(50)
);

CREATE TABLE Objects (
    id        INT PRIMARY KEY,
    object    VARCHAR(50),
    type_id   INT,
    measure   INT,
    config_id INT
);
