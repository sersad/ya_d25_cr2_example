### 📋 Формулировка 2 задачи
В файле `departments.csv` записаны сведения о сотрудниках отделов с заголовками (разделитель — точка с запятой):
`no;department;employee;specialization;experience;budget`

Для каждого **отдела** определите:
1. Наибольший (лексикографически) профиль специализации (`specialization`).
2. Максимальный опыт работы в годах (`experience`).
3. Суммарный выделенный бюджет (`budget`).

Запишите сведения в файл `report.json` в виде словаря, где ключи — названия отделов, а значения — словари с ключами `specialization`, `experience`, `budget`.

---

### 🧪 Варианты входных данных и ожидаемый результат

#### Вариант 1
**Ввод (`departments.csv`):**
```csv
no;department;employee;specialization;experience;budget
1;R&D;Alice;AI;5;5000
2;R&D;Bob;Backend;8;6000
3;Marketing;Charlie;SEO;3;2000
4;R&D;Dave;AI;10;4000
5;Marketing;Eve;Content;4;3000
```
**Вывод (`report.json`):**
```json
{
    "Marketing": {
        "budget": 5000,
        "experience": 4,
        "specialization": "SEO"
    },
    "R&D": {
        "budget": 15000,
        "experience": 10,
        "specialization": "Backend"
    }
}
```
